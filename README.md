# Astro_mi_primer_blog

En este tutorial, aprenderás las características clave de Astro creando un blog completamente funcional, ¡desde cero hasta el lanzamiento completo! 

Este proyecto se creó a partir del template `minimal` de Astro. A continuación hay información útil para empezar y ejecutar el proyecto localmente.

## Estructura del proyecto

```
/
 public/
 src/
    pages/
        index.astro
 package.json
```

Coloca archivos `.astro` o `.md` en `src/pages/` para que se conviertan en rutas.

## Comandos principales

Ejecuta los comandos desde la raíz del proyecto:

| Comando | Acción |
| :--- | :--- |
| `npm install` | Instala dependencias |
| `npm run dev` | Inicia servidor de desarrollo (por defecto en http://localhost:3000) |
| `npm run build` | Construye el sitio para producción en `./dist/` |
| `npm run preview` | Previsualiza la build localmente |

## Recursos

- Documentación Astro: https://docs.astro.build
